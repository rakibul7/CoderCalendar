This is a vanilla Flask app.

### Instructions

#### Installation

```bash
# Setup a virtual env
$ virtualenv env

# Activate it
$ source env/bin/activate

# Install dependencies
(env) $ pip install -r requirements.txt
```

#### Running the app

```bash
(env) $ python route.py
```

#### Running the tests

```bash
(env) $ python tests.py
```

Don't forget the deactivate the virtualenv once you're done!
```bash
(env) $ deactivate
```
